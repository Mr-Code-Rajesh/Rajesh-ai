# Rajesh P | The Neural Protocol - AI Product Portfolio

![Neural OS Interface](https://raw.githubusercontent.com/rajesh-ai/portfolio/main/public/og-preview.png)

A high-performance, ultra-premium developer portfolio engineered for the product-first era. This project implements a sophisticated **"Neural OS"** aesthetic, combining cutting-edge frontend technologies with advanced SEO and interactive AI systems.

## 💎 Core Features

### 1. **Neural OS Aesthetic**
- **Neural Boot Sequence**: A high-end preloader simulating a system initialization on first session load.
*   **Interactive Background**: A custom Canvas-based particle system with realistic mouse repulsion physics.
- **Cyber-HUD Interface**: A cohesive design system utilizing **Cyber-Blue (#00D1FF)** and **Matrix-Green (#00FF41)** accents.
- **System Feed**: Real-time simulated terminal logs integrated into the background for immersive storytelling.

### 2. **Advanced SEO Architecture**
- **Structured Data**: Integrated Schema.org (JSON-LD) defining the identity as a **Senior AI Product Engineer**.
- **Section-Specific Optimization**: Tailored metadata and semantic HTML (`sr-only` content) for deep indexing.
- **OpenGraph & Twitter Cards**: Fully optimized for social sharing with preview images.

### 3. **High-Performance Infrastructure**
- **Tech Matrix**: Built on **Next.js 16** and **React 19** with **Tailwind CSS v4**.
- **Smooth Interaction**: Global micro-interactions, magnetic buttons, and scroll-triggered entrance animations powered by **Framer Motion**.
- **Case Study Dashboard**: A technical deep-dive system for showcasing complex projects with "Problem/Solution/Impact" metrics.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/rajesh-panda/ai-portfolio.git
cd ai-portfolio/ai-me
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Setup
Create a `.env.local` file in the root directory and add the following keys:
```env
# EmailJS Config (Client-side)
NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key

# Database
MONGODB_URI=your_mongodb_connection_string

# Resend / OpenAI (Optional for advanced features)
RESEND_API_KEY=your_resend_api_key
OPENAI_API_KEY=your_openai_api_key
```

### 4. Launch Development Server
```bash
npm run dev
```
Navigate to [http://localhost:3000](http://localhost:3000) to initialize the protocol.

## 📂 Project Structure

- `/app`: Next.js App Router for routing and layout.
- `/components`: Modular, reusable UI components (Hero, TechStack, Navbar, etc.).
- `/data`: JSON stores for projects, skills, and resume data.
- `/hooks`: Custom React hooks for global patterns (e.g., Command Palette).
- `/public`: Static assets, OG images, and icons.
